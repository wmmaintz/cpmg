About The Website

Technical information regarding this website's construction.

Responsive Design

This website was constructed with HTML5, CSS3, and Javascript to achieve a "responsive design" and was built upon an AngularJS Single-Page Application (SPA) framework. It also employs Twitter's bootstrap which is "the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web." This allows the website to be viewed on multiple devices, from large desktop screens to mobile devices like tablets and smartphones. The cascading style sheets were created using LESS, a pre-compiler for generating Cascading Style Sheet (CSS) files, and all menu items and social media links utilize Glyphicons.

Single Page Application

This website uses a Single Page Application design, meaning that all content is presented using a single HTML file as its base and sections of that base page are swapped out/replaced by other content that the user has selected to see. This eliminates the need to repaint the entire screen in order to load multiple HTML pages everytime the user moves from one page to another. Using the AngularJS framework and its User Interface extension, Angular-UI, this website takes the standard Model-View-Controller (MVC) pattern and extends it, employing a Model-View-Whatever design, including dependency injection, directives, two-way data binding, controllers, services, and filters.
